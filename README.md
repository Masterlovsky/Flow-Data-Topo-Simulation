# flow_data_topo_simulation
> 使用第三方库pyecharts创建节点流关系图
> pyecharts官网： https://pyecharts.org/#/

### 使用说明：

1. 将拓扑文件放入community_small.txt中，\t分隔
2. 将节点负载和边负载关系放入flow_data.txt中，空格分隔
3. 运行simulation_flow_graph.py 生成html文件
4. 在浏览器打开html文件预览

### 环境依赖：
- pyecharts 1.9.0
- numpy 1.19.2
